# Smart Campus Resource Management System

## 📌 Problem Statement
In college campuses, rooms such as classrooms, labs, and halls are shared resources.
Manual booking leads to clashes, double bookings, and poor visibility of availability.

## 💡 Solution
This project provides a centralized system to:
- Manage campus rooms
- Book rooms for specific time slots
- Prevent overlapping bookings automatically
- View real-time room availability

## ⚙️ Tech Stack
**Backend**
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- REST APIs

**Frontend**
- React (Vite)
- React Router
- HTML, CSS, JavaScript

## 🔑 Key Features
- Full CRUD operations for rooms
- Time-slot based room booking
- Conflict detection for overlapping bookings
- RESTful API architecture
- Full-stack integration (React + Spring Boot)

## 🚀 How to Run

### Backend
1. Clone repository
2. Configure MySQL in `application.properties`
3. Run Spring Boot application

### Frontend
```bash
npm install
npm run dev
